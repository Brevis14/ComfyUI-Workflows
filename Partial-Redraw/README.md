# 🧩 Partial-Redraw

### 🌟 功能简介  
该工作流可对图像的 **指定区域进行遮罩重绘**，保持其他部分不变，实现内容替换、细节修复或背景调整。  
适用于：图像修补、指定区域替换、风格化修复等任务。

---

### 🧠 原理概述  
- 通过 **Mask 区域** 控制重绘范围；  
- 使用 **ControlNet (Canny)** 保持全局结构一致性；  
- **KSampler** 控制重绘强度；  
- 支持文本提示词进行引导生成。

---

### 🧱 核心节点  
| 模块 | 节点 | 功能说明 |
|------|------|-----------|
| 图像输入 | LoadImage | 导入原图 |
| 遮罩 | LoadMask / 手绘 Mask | 指定重绘区域 |
| 结构保持 | ControlNet (Canny) | 保持边缘一致性 |
| 生成 | KSamplerAdvanced | 控制重绘强度与风格 |
| 输出 | VAEDecode + SaveImage | 输出最终图像 |

---

### ⚙️ 使用步骤  
1. 导入原图。  
2. 上传或绘制遮罩区域。  
3. 在提示词中描述你想要替换的内容。  
4. 点击运行生成。  
5. 输出结果会保存在 `ComfyUI/output`。

---

### 💡 提示建议  
- `denoise` 参数越低 → 原图保留越多；越高 → 重绘范围越大。  
- 遮罩边缘建议羽化（Feather Mask）以获得更自然融合。
